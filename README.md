# OnlineFoodOrderingSystem-OOP-FINAL
Java console application developed as an Object-Oriented Programming final project.
## Project Description
This project is a Java console-based Online Food Ordering System developed 
to demonstrate Object-Oriented Programming principles such as encapsulation,
inheritance, interfaces, and polymorphism.

## Features
- User login authentication
- Restaurant listing
- Menu display
- Add items to cart
- Discount coupon support
- Payment selection (Cash / Credit Card)
- Order summary and rating system

## Technologies Used
- Java
- JUnit 5
- Git & GitHub
